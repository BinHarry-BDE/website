# BinHarry-Website

Site Web pour l'association BinHarry à l'IUT de reims dans le département informatique

## Prérequis

- **Node.js** v24.11.1 (ou compatible) - [installer nvm](https://github.com/nvm-sh/nvm)
- **npm** (inclus avec Node.js)
- **Git**
- **Compte Cloudflare** (pour déploiement)

### Vérifier vos versions

```bash
node --version  # doit être v24.11.1
npm --version
```

## Stack Technique

- Next.js
- React 19
- OpenNext
- Cloudflare Workers
- TypeScript
- Tailwind CSS

## Développement

Le projet est développé par les membres du BDE mais vous pouvez demandé en privé de participer a son developpement.
Pour les futurs membres du BDE un fichier de note vous est adressé dans le dossier `public/docs`

### Démarrer le projet

```bash
npm install
npm run dev
```

Et vous êtes prêt ! 🚀
